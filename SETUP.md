# MineScope — Setup Guide

## Files Not Included in Repository

The following large files/directories are excluded from Git and must be set up locally.

### 1. Metagenomics FASTQ (1.1 GB)

Download from NCBI SRA:
```bash
# Using SRA Toolkit
fasterq-dump SRR6189722 -O data/bronze/metagenomics/

# Or download FASTQ directly from NCBI and place at:
# data/bronze/metagenomics/SRR6189722.fastq
```

### 2. MetaPathways Reference Databases (~1 GB)

The `databases/` directory contains formatted reference databases for MetaPathways v3.5.

To rebuild from scratch:
```bash
# 1. Download SwissProt
mkdir -p databases/functional
curl -o databases/functional/swissprot.gz https://ftp.uniprot.org/pub/databases/uniprot/current_release/knowledgebase/complete/uniprot_sprot.fasta.gz
gunzip databases/functional/swissprot.gz

# 2. Format with BLAST (inside MetaPathways container)
docker run --rm --platform linux/amd64 \
  -v $(pwd)/databases:/databases \
  quay.io/hallamlab/metapathways \
  bash -c "export PATH=/opt/conda/envs/metapathways/bin:\$PATH && mkdir -p /databases/functional/formatted && makeblastdb -in /databases/functional/swissprot -dbtype prot -parse_seqids -out /databases/functional/formatted/swissprot"

# 3. Create names file
docker run --rm --platform linux/amd64 \
  -v $(pwd)/databases:/databases \
  quay.io/hallamlab/metapathways \
  bash -c "grep '^>' /databases/functional/swissprot > /databases/functional/formatted/swissprot-names.txt"

# 4. Download SILVA (taxonomic)
curl -O https://www.arb-silva.de/fileadmin/silva_databases/release_138.1/Exports/SILVA_138.1_LSURef_NR99_tax_silva_trunc.fasta.gz
curl -O https://www.arb-silva.de/fileadmin/silva_databases/release_138.1/Exports/SILVA_138.1_SSURef_NR99_tax_silva_trunc.fasta.gz
gunzip *.gz
mv SILVA_138.1_*trunc* databases/taxonomic/

# 5. Download enzyme.dat
curl -o databases/functional_categories/enzyme.dat https://ftp.expasy.org/databases/enzyme/enzyme.dat

# 6. Download NCBI taxonomy
curl -o databases/ncbi_tree/new_taxdump.tar.gz https://ftp.ncbi.nlm.nih.gov/pub/taxonomy/new_taxdump/new_taxdump.tar.gz
tar xzf databases/ncbi_tree/new_taxdump.tar.gz -C databases/ncbi_tree/
```

### 3. Node Modules

```bash
cd dashboard
npm install
```

### 4. Python Virtual Environment

```bash
uv sync
```

### 5. Nextflow Results

Pipeline results are generated by running:
```bash
nextflow run main.nf -profile docker \
  --input_fastq data/bronze/metagenomics/SRR6189722.fastq \
  --outdir results_full
```

Then run the data pipeline:
```bash
uv run python scripts/run_pipeline.py
```

## Quick Start (after setup)

```bash
# 1. Generate synthetic data (if not already present)
uv run python scripts/data_generation/generate_lidar.py
uv run python scripts/data_generation/generate_chemistry.py

# 2. Run data pipeline (bronze → silver → gold → dashboard JSON)
uv run python scripts/run_pipeline.py

# 3. Start dashboard
cd dashboard
npm run dev
```

## Database Layout (expected by MetaPathways)

```
databases/
├── functional/
│   ├── swissprot
│   └── formatted/
│       ├── swissprot.* (BLAST index files)
│       └── swissprot-names.txt
├── taxonomic/
│   ├── SILVA_138.1_LSURef_NR99_tax_silva_trunc
│   ├── SILVA_138.1_SSURef_NR99_tax_silva_trunc
│   └── formatted/ (BLAST index files)
├── ncbi_tree/
│   └── ncbi_taxonomy_tree.txt
└── functional_categories/
    └── enzyme.dat
```
